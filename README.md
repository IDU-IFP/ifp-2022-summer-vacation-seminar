# 👨🏻‍🏫 ifp-2022-summer-vacation-seminar
IFP 2022년 여름방학 HTML &amp; CSS 세미나

## ⚠ 과제 제출 방법
해당 레파지토리를 fork한 후, main브랜치의 본인의 이름으로 폴더를 만들고, 각 세미나 회차에 대한 폴더 안에서 코드 작업을 한 후 pr을 보내주세요.
merge는 코드리뷰 후 세미나 진행자가 할 예정이니 merge는 하지 말아주세요.
